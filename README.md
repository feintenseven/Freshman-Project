# Freshman-Project

## Course Information
- **Course**: Introduction to Computing (C)
- **University**: Peking University
- **Project Type**: Final Assignment

## Project Overview
This project explores web crawling, data analysis, and visualization with LLM assistance, focusing on the "Cai Li Ba" forum. The aiotieba tool was used for data acquisition, followed by comprehensive cleaning, analysis, and visualization processes.

## Key Features
- LLM-assisted web crawling implementation
- Multi-dimensional data analysis of forum content
- Automated data cleaning pipeline
- Comprehensive visualization outputs

## Tools & Technologies
- **Data Acquisition**: aiotieba (by lumina37)
- **Analysis & Visualization**: Python
